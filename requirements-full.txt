Executive Summary
This document provides the complete, final specification for the Construction Marketplace Platform. It includes all workflows, features, payment systems, and technical requirements needed to build and launch the MVP.
Platform Overview

The Construction Marketplace Platform is a web-based marketplace that connects:

USER TYPES:
    • Project Owners (Individuals & Companies)
      - Create and publish projects (DESIGN only, CONSTRUCTION only, or DESIGN + CONSTRUCTION)
      - Receive and compare bids from contractors and engineers
      - Request optional supervision services
      - Order professional contract drafting
      - Execute projects through shared workspace
    
    • Contractors (Companies Only)
      - Purchase project access (Krasat - 100 SAR)
      - Ask project owners questions
      - Submit bids on construction projects
      - Execute awarded projects
      - Earn reputation through ratings
    
    • Engineers (Individuals & Companies)
      - Purchase project access (Krasat - 100 SAR)
      - Ask project owners questions
      - Submit bids on design or supervision projects
      - Choose specialization: Designer OR Supervisor OR Both
      - Provide professional engineering services
      - Maintain wallet balance (500 SAR) for supervision eligibility
      - Earn reputation through ratings
    
    • Admin
      - Verify contractor and engineer credentials
      - Review and approve projects
      - Handle disputes and complaints
      - Manage refund requests
      - Monitor platform activity

CORE FEATURES:
    ✓ 3 Project Types (Design, Construction, Design + Construction)
    ✓ Smart bid ranking (100-point algorithm)
    ✓ AI profile ranking and badges
    ✓ Krasat system (100 SAR project access + Q&A)
    ✓ Execution Workspace (shared communication & files)
    ✓ Upselling 1: Supervision Service (100 SAR request + 500 SAR wallet)
    ✓ Upselling 2: Professional Contract (150 SAR)
    ✓ Digital signatures and contract storage
    ✓ Comprehensive rating and review system
    ✓ Secure payment processing

REVENUE STREAMS:
    1. Krasat Access: 100 SAR per project (contractors/engineers)
    2. Supervision Request: 100 SAR per request (project owners)
    3. Supervision Wallet: 500 SAR balance requirement (engineers)
    4. Professional Contract: 150 SAR per contract (project owners)

2. Project Types & Categories

THREE PROJECT TYPES:

TYPE 1: DESIGN ONLY
    Purpose: Projects requiring only design/planning work
    Examples:
        • Architectural design
        • Engineering calculations & drawings
        • Interior design planning
        • 3D rendering
        • Structural analysis
        • MEP (Mechanical, Electrical, Plumbing) design
    
    Who can bid: Engineers with "Designer" specialty ONLY
    Visible to: Engineers ONLY (not contractors)
    Bidder qualifications:
        • Verified engineer status
        • Designer specialization selected
        • Relevant engineering discipline
        • Good track record (optional)
    
    Workflow:
        1. Owner creates DESIGN_ONLY project
        2. Engineers see project
        3. Engineers bid with design solutions
        4. Owner compares designs
        5. Owner awards engineer
        6. Engineer executes design work
        7. Deliverables: drawings, specifications, 3D models

TYPE 2: CONSTRUCTION ONLY
    Purpose: Projects requiring only construction/execution work
    Examples:
        • Building construction
        • Renovations
        • Interior finishing
        • Installation work
        • Repairs & maintenance
        • Painting & decoration
    
    Who can bid: Contractors (Companies) ONLY
    Visible to: Contractors ONLY (not engineers)
    Bidder qualifications:
        • Verified contractor status
        • Relevant trade/specialty
        • Service area includes location
        • Good track record (optional)
    
    Workflow:
        1. Owner creates CONSTRUCTION_ONLY project
        2. Contractors see project
        3. Contractors bid with execution plans
        4. Owner compares bids
        5. Owner awards contractor
        6. Contractor executes work
        7. Deliverables: completed work, photos, sign-off

TYPE 3: DESIGN + CONSTRUCTION
    Purpose: Complete projects requiring both design and construction
    Examples:
        • Full house renovation (design + build)
        • New building project
        • Complex renovation
        • Custom construction
    
    Who can bid: Contractors (Companies) ONLY
    Visible to: Contractors ONLY (not engineers)
    Bidder options:
        • Contractor bids for construction phase only
        • Engineer bids for design phase only
        • Either can bid for full scope
    
    Workflow:
        1. Owner creates DESIGN_AND_CONSTRUCTION project
        2. Both contractors and engineers see project
        3. Options for owner:
            A. Award contractor for construction only
            B. Award engineer for design only
            C. Award both (collaborate)
            D. Award contractor for full scope
            E. Award engineer for full scope
        4. Execution per agreement
        5. Can use supervision for either/both phases

3. Project Owner Complete Workflow

COMPLETE PROJECT OWNER JOURNEY (12 STAGES):

STAGE 1: REGISTRATION & PROFILE
    ├─ Register account
    ├─ Verify email/phone
    ├─ Complete profile:
    │  ├─ Name, company (if applicable)
    │  ├─ Contact information
    │  ├─ Profile image
    │  ├─ Bio/description
    │  └─ Project preferences
    └─ Status: ACTIVE (ready to create projects)

STAGE 2: PROJECT CREATION
    ├─ Click "Create New Project"
    ├─ Project Type Selection:
    │  ├─ DESIGN_ONLY
    │  ├─ CONSTRUCTION_ONLY
    │  └─ DESIGN_AND_CONSTRUCTION
    ├─ Enter project details:
    │  ├─ Title
    │  ├─ Category
    │  ├─ Location (city, district)
    │  ├─ Budget range
    │  ├─ Detailed description (500+ chars)
    │  └─ Start date
    ├─ Upload files:
    │  ├─ Project images (5-10)
    │  ├─ Drawings/designs (if any)
    │  ├─ BOQ (Bill of Quantities)
    │  ├─ Site photos
    │  └─ Any specifications
    └─ Save as DRAFT (auto-saved)

STAGE 3: ADMIN REVIEW
    ├─ Project submitted for review
    ├─ Status: PENDING_REVIEW
    ├─ Admin checks:
    │  ├─ Completeness
    │  ├─ File quality
    │  ├─ Compliance
    │  ├─ Budget reasonableness
    │  └─ No policy violations
    ├─ Admin approves OR requests modifications
    └─ Owner notified (email + in-app)

STAGE 4: PROJECT PUBLISHED
    ├─ Status: PUBLISHED
    ├─ Visible to appropriate users:
    │  ├─ DESIGN_ONLY → Engineers only
    │  ├─ CONSTRUCTION_ONLY → Contractors only
    │  └─ DESIGN_AND_CONSTRUCTION → Both
    ├─ Smart matching activated
    ├─ Notifications sent to matching users
    └─ Bidding window opens (30 days default)

STAGE 5: RECEIVE BIDS
    ├─ Contractors/engineers submit bids
    ├─ Bids automatically ranked by AI
    ├─ Real-time notifications when bids arrive
    ├─ Owner can view:
    │  ├─ All bids ranked by score
    │  ├─ Bidder profile & ratings
    │  ├─ Proposed price & timeline
    │  ├─ Detailed proposal
    │  └─ Supporting documents
    └─ Status: BIDDING (active)

STAGE 6: KRASAT & QUESTIONS
    ├─ Each bidder paid 100 SAR (already done)
    ├─ Owner sees notes from bidders
    ├─ Owner can ask follow-up questions:
    │  ├─ Click on bid → "Ask Question"
    │  ├─ Message bidder directly
    │  ├─ Bidder responds via platform
    │  └─ Full conversation history
    ├─ Owner can clarify requirements
    └─ Bidders can request details

STAGE 7: COMPARE & SHORTLIST
    ├─ View all ranked bids
    ├─ Can shortlist up to 3 bidders
    ├─ Compare side-by-side:
    │  ├─ Price & timeline
    │  ├─ Bidder ratings & reviews
    │  ├─ Experience & badges
    │  ├─ Proposal quality
    │  └─ Risk assessment
    ├─ Ask detailed questions
    ├─ Request bid modifications
    └─ Narrow down to best option

STAGE 8: AWARD CONTRACTOR/ENGINEER
    ├─ Select preferred bidder
    ├─ Click "Award This Bid"
    ├─ Confirmation dialog
    ├─ Confirm award
    ├─ Status: AWARDED
    ├─ Other bids auto-rejected
    ├─ Bidders notified
    └─ Execution workspace created

STAGE 9: UPSELLING - SUPERVISION SERVICE (Optional)
    ├─ "Would you like professional supervision?"
    ├─ Cost: 100 SAR (one-time fee)
    ├─ If yes:
    │  ├─ Pay 100 SAR
    │  ├─ System broadcasts to eligible engineers
    │  ├─ Engineers with 500+ SAR wallet can bid
    │  ├─ Engineers submit supervision bids
    │  ├─ Bids ranked by AI
    │  ├─ Owner compares supervision proposals
    │  ├─ Owner awards supervisor
    │  ├─ 500 SAR deducted from supervisor wallet
    │  └─ Supervisor joins execution workspace
    └─ If no: Skip to Stage 10

STAGE 10: UPSELLING - PROFESSIONAL CONTRACT (Optional)
    ├─ "Choose contract type"
    ├─ Option A: SIMPLE (free)
    │  ├─ Standard template
    │  ├─ Basic terms
    │  └─ Auto-filled data
    ├─ Option B: PROFESSIONAL (150 SAR)
    │  ├─ Custom drafted
    │  ├─ All project files attached
    │  ├─ Comprehensive terms
    │  ├─ Quality standards detailed
    │  └─ Digital signatures
    ├─ If PROFESSIONAL:
    │  ├─ Pay 150 SAR
    │  ├─ Contract auto-generated
    │  ├─ Both parties review
    │  ├─ Both parties sign digitally
    │  └─ Contract finalized
    └─ Status: CONTRACT_SIGNED

STAGE 11: EXECUTION WORKSPACE
    ├─ Shared work area with contractor/engineer
    ├─ Contains:
    │  ├─ Direct messaging
    │  ├─ File sharing & version control
    │  ├─ Milestone tracking
    │  ├─ Daily progress updates
    │  ├─ Photo documentation
    │  ├─ Issue/problem reporting
    │  ├─ Contract reference
    │  └─ Payment tracking
    ├─ Contractor/engineer executes work
    ├─ Supervisor monitors (if assigned)
    ├─ Regular status updates
    ├─ Owner provides feedback
    └─ Project progresses through milestones

STAGE 12: COMPLETION & RATING
    ├─ Project marked complete
    ├─ Final inspection/approval
    ├─ Contract marked COMPLETED
    ├─ Owner submits rating:
    │  ├─ Stars (1-5)
    │  ├─ Review comments
    │  ├─ Overall satisfaction
    │  └─ Recommendation
    ├─ Rating published on contractor profile
    ├─ Project archived
    └─ PROCESS COMPLETE
